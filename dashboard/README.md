# Dashboard MVP

Next.js App Router dashboard for buyer operations.

## Setup

1. Copy `.env.example` to `.env.local`.
2. Set `NEXT_PUBLIC_API_BASE_URL`.
3. Install and run:
   - `npm install`
   - `npm run dev`

## Pages

- `/profile` - buyer registration and profile fetch.
- `/sessions` - list buyer sessions.
- `/nodes` - list active nodes and node stats.
- `/topup` - MVP top-up action.
